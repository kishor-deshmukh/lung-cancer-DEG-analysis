# Lung Cancer DEG Analysis

## Overview
Differential gene expression analysis to identify key 
oncogenes in lung cancer progression using public GEO dataset.

Internship project — IIT Bombay, under Prof. Rohit Shrivastava

## Dataset
- Source: GEO Database (NCBI)
- Condition: Lung Cancer Tumor vs Normal tissue
- Type: RNA-seq gene expression data

## Methods
- Differential Expression: DESeq2
- Cutoff: padj < 0.05 | log2FC > 1
- Pathway Enrichment: GO + KEGG
- Visualization: ggplot2, pheatmap

## Key Results
- Identified significant upregulated and downregulated genes
- Volcano plot showing DEG distribution
- Heatmap of top differentially expressed genes
- GO biological process enrichment
- KEGG cancer pathway enrichment

## Tools & Languages
Python | R | DESeq2 | ggplot2 | pheatmap | clusterProfiler
